# План синхронизации внешней документации с SIFS-Theory-Core

Рекомендации для проектов и документов, которые ссылаются на SIFS Theory (Spacetime) или используют то же математическое ядро (k, W(n), φ, FIB). Основано на анализе документации из внешнего проекта (см. [external-projects-documentation-analysis.md](external-projects-documentation-analysis.md)).

---

## Фаза 1: Связь и терминология (приоритет)

| Шаг | Действие |
|-----|----------|
| 1.1 | Добавить раздел «Связь с SIFS Theory (Spacetime)» со ссылками на репозиторий [SIFS-Theory-Core](https://github.com/m0rfy/SIFS-Theory-Core) и ключевые документы (docs, White Paper, Terminology Guide). |
| 1.2 | Заменить «подтверждено»/«доказано» на **«согласуется с данными»** / **«предсказано теорией»** в наблюдательном контексте. |
| 1.3 | Добавить подраздел «Интерпретация vs подтверждение» в разделы с экспериментальными утверждениями. |
| 1.4 | Унифицировать термины: «масштабная координата S», «RS-warping», при микро-объектах — «effective horizon» (см. [Terminology Guide](../terminology-guide.md)). |

---

## Фаза 2: Константы и отличия

| Шаг | Действие |
|-----|----------|
| 2.1 | Явно указать: в документе по **приложениям** k = 1/π² — безразмерная константа; в SIFS Theory (Spacetime) k ≈ 0.1 M_Pl — размерный параметр. |
| 2.2 | В сноске к RS-метрике: дискретное n (уровни 0..9) в приложениях vs непрерывное S в физической теории. |

## Фаза 3: Структура и приложения

| Шаг | Действие |
|-----|----------|
| 3.1 | В разделе про приложения (например финансы) добавить отсылку к реализации (SIFS Live, Freqtrade и т.д.) и к технической документации, откуда берутся цифры (K, веса, порог 55, ATR, φ-цели). |
| 3.2 | При необходимости перегруппировать оглавление: Введение → Основания → Ядро → Уровни → Приложения → Верификация и ограничения → Заключение и ссылки. |

## Чего не делать

- Не смешивать физические предсказания Theory-Core (DESI, протон, KK-моды) в основной текст документа по математическому ядру и приложениям — только ссылки.
- Не менять выведенные формулы (k = 1/π², W(n), V_th(n)) без согласования с теорией.
- Не дублировать White Paper или theory/ целиком — только ссылки и краткое описание связи.
