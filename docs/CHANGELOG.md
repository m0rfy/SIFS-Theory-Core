## Формат версионирования

Версии документов следуют формату [Semantic Versioning](https://semver.org/):

- **MAJOR** - Несовместимые изменения (изменение структуры, удаление разделов)
- **MINOR** - Новые функции (добавление разделов, новые документы)
- **PATCH** - Исправления ошибок (опечатки, исправления формул)

## История изменений терминологии

### 2025-01-27: Миграция терминологии

**Замены:**
- "черная дыра" (микро) → "effective horizon" / "horizon-like trapped region"
- "подтверждено данными" → "согласуется с данными"
- "доказано" → "предсказано теорией"
- "экспериментальное подтверждение" → "наблюдательное согласие"

**Причина:** Обеспечение академической строгости и предотвращение концептуальной путаницы.

**Документы:** См. [Migration Guide](migration-guide.md) для полного списка изменений.
