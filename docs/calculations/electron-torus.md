# Расчёт модели электрона как тора

Детальные расчёты модели электрона как кольцевой сингулярности Керра-Ньюмана.

## Параметры электрона

### Наблюдаемые параметры

- **Масса:** m_e = 9.109×10⁻³¹ кг
- **Заряд:** Q = e = 1.602×10⁻¹⁹ Кл
- **Спин:** J = ℏ/2 = 5.273×10⁻³⁵ Дж·с
- **Радиус Комптона:** λ_C = ℏ/(m_e c) = 3.862×10⁻¹³ м

## Метрика Керра-Ньюмана

### Параметры КН-решения

Для электрона:
- **Масса:** M = m_e
- **Заряд:** Q = e
- **Угловой момент:** a = J/M = ℏ/(2m_e)

### Радиус кольца

Радиус кольцевой сингулярности:

```
a = ℏ/(2m_e) = λ_C/2 = 1.931×10⁻¹³ м
```

Это соответствует половине Комптоновской длины волны.

### Условие горизонта

Условие существования горизонта:

```
M² ≥ Q² + a²
```

Проверка для электрона:

**В геометрических единицах (G = c = 1):**
- M = m_e = 9.109×10⁻³¹ кг = 4.185×10⁻⁶⁵ м
- Q = e/√(4πε₀) = 1.380×10⁻³⁶ м
- a = ℏ/(2m_e) = 1.931×10⁻¹³ м

**Сравнение:**
- M² ≈ 1.75×10⁻¹²⁹ м²
- Q² ≈ 1.90×10⁻⁷² м²
- a² ≈ 3.73×10⁻²⁶ м²

**Результат:**
```
M² << a²
```

Условие горизонта **катастрофически нарушается**. Горизонта нет — это голая сингулярность.

## Гиромагнитное отношение

### Классическое значение

Для точечной частицы: g = 2

### Расчёт через геометрию Керра-Ньюмана

Магнитный момент:

```
μ = (e/2m_e) × J = (e/2m_e) × (ℏ/2) = eℏ/(4m_e)
```

Гиромагнитное отношение:

```
g = 2μ/(μ_B) = 2
```

где μ_B = eℏ/(2m_e) — магнетон Бора.

### Поправки КЭД

Квантовые поправки дают:

```
g = 2 + (α/2π) + ...
```

где α ≈ 1/137 — постоянная тонкой структуры.

**Интерпретация в SIFS:**
Поправка (α/2π) = проекция энтропии запутанности горизонта на 4D-брану:

```
(α/2π) = S_entanglement / S_max
```

## Стабильность тора на границе 3D-4D

### Топологическая стабильность

Кольцевая сингулярность стабильна, потому что:
- Топологический дефект пространства-времени
- Замкнутая струна (в низкоэнергетическом пределе)
- Не может распасться без нарушения топологии

### Регуляризация

Внутри радиуса a метрика регуляризуется:
- Сингулярность вытесняется из физической области
- Bag Model: метрика становится плоской или де-Ситтеровской
- Устраняет CTC и голую сингулярность

### ER=EPR стабильность

Геометрия запутанности обеспечивает стабильность:
- Виртуальные фотоны = геометрия червоточины в 5D
- Кольцевая сингулярность = горловина кротовой норы
- Запутанность обеспечивает квантовую стабильность

## Заключение

Расчёты показывают, что:
- Электрон как КН-решение имеет голую сингулярность (нет горизонта)
- Радиус кольца соответствует Комптоновской длине волны
- Гиромагнитное отношение g = 2 возникает из чистой геометрии
- Поправки КЭД интерпретируются через энтропию запутанности
- Стабильность обеспечивается топологией и ER=EPR
