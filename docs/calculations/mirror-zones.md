# Расчёт зеркальных зон (n < 0)

Детальные расчёты условий образования зеркальных зон и топологической инверсии.

## Условия n² < 0

### Критическое условие

Показатель преломления становится отрицательным при:

```
n²(r, S) < 0
```

### Связь с натяжением браны

Показатель преломления связан с натяжением:

```
n²(r, S) = 1 + (σ(r, S) / σ_crit) × f(r, S)
```

При σ → σ_crit:
- n² → 0
- n² < 0 при σ > σ_crit

### Условие критичности

```
σ(r, S) ≥ σ_crit
```

где σ_crit ≈ 10²⁴ Дж/нм² = 10³³ Дж/м²

## Топологическая инверсия

### Инверсия времени

При n² < 0 вектор времени меняет направление:

```
dt → −dt
```

**Физический смысл:**
- Прошлое и будущее меняются местами
- Время течёт в обратном направлении
- Создаёт условия для антиматерии

### Инверсия заряда

Положительный заряд → отрицательный:

```
Q → −Q
```

**Физический смысл:**
- Частицы превращаются в античастицы
- Заряд инвертируется топологически
- Образуются пары частиц

### Инверсия хиральности

Левая хиральность → правая:

```
ψ_L → ψ_R
```

**Физический смысл:**
- Нарушение четности
- Топологический выворот метрики
- Зеркальная структура

## Образование антиматерии

### Механизм рождения пар

При разрыве вакуума (σ ≥ σ_crit):

```
Вакуум → q + q̄ (кварк + антикварк)
```

Энергия для рождения пары:

```
E_pair = 2m_q c² = σ_crit × A_critical
```

### Критическая площадь

Для кварка (m_q ≈ 5 МэВ):

```
A_critical = (2 × 5×10⁶ × 1.602×10⁻¹⁹) / (10³³)
           = 1.602×10⁻¹² / 10³³
           = 1.602×10⁻⁴⁵ м²
```

### Условия образования

Зеркальные зоны образуются в точках:
- Критического напряжения браны: σ ≥ σ_crit
- Экстремальной кривизны пространства-времени
- Высокой плотности энергии (близко к планковской)

## Энергия извлечения из зон с n < 0

### Потенциальная энергия деформации

Энергия, запасённая в деформации браны:

```
E_deformation = ∫_{A_zone} (σ − σ_crit) dA
```

где A_zone — площадь зеркальной зоны.

### Топологическая энергия инверсии

Энергия, связанная с топологической инверсией:

```
E_topological = (σ_crit × A_inversion) × f(S)
```

где:
- A_inversion — площадь инвертированной области
- f(S) — функция, учитывающая фрактальную структуру

### Общая энергия извлечения

```
E_extraction = E_deformation + E_topological
```

### Численный пример

Для зеркальной зоны площадью A = 10⁻⁵⁰ м²:

```
E_deformation = (σ − σ_crit) × A
              = 0.1×σ_crit × 10⁻⁵⁰
              = 0.1 × 10³³ × 10⁻⁵⁰
              = 10⁻¹⁸ Дж
              = 6.24 эВ
```

## Математический формализм

### Уравнение натяжения

Локальное натяжение как функция координат:

```
σ(r, S) = σ₀ + δσ(r, S)
```

где:
- σ₀ — фоновое натяжение
- δσ — локальное отклонение

### Критическое условие

```
σ₀ + δσ_crit = σ_crit
```

Откуда:

```
δσ_crit = σ_crit − σ₀
```

### Показатель преломления

Связь натяжения и показателя преломления:

```
n²(r, S) = 1 + (σ(r, S) / σ_crit) × f(r, S)
```

При σ → σ_crit:
- n² → 1 + f(r, S)
- n² < 0 при f(r, S) < −1

## Физические следствия

### Рождение частиц

При разрыве вакуума рождаются пары:
- Кварк-антикварк (адронизация)
- Электрон-позитрон
- Другие пары частиц

### Стабильность материи

Обычная материя стабильна, потому что:
- Натяжение браны ниже критического
- n² > 0 (положительный показатель преломления)
- Нет условий для разрыва вакуума

### Экстремальные условия

В экстремальных условиях (близко к планковской энергии):
- Натяжение может достигать σ_crit
- Образуются зеркальные зоны
- Возможен разрыв вакуума

## Заключение

Расчёты показывают, что:
- Зеркальные зоны (n < 0) образуются при достижении критического натяжения
- Топологическая инверсия создаёт условия для антиматерии
- Энергия извлечения связана с потенциальной энергией деформации
- Механизм рождения пар частиц объясняется разрывом вакуума
