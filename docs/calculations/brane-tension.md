# Расчёт критического напряжения браны

Детальные расчёты критического напряжения браны и энергии прокола.

## Критическое напряжение

### Определение

Критическое напряжение браны:

```
σ_crit ≈ 10²⁴ Дж/нм²
```

### Физический смысл

При достижении этого значения:
- Происходит разрыв вакуума
- Рождаются пары частиц (адронизация)
- Показатель преломления становится отрицательным (n² < 0)

## Расчёт из первых принципов

### Энергия деформации

Энергия деформации браны:

```
E_deformation = σ × ΔA
```

где:
- σ — натяжение браны
- ΔA — изменение площади деформации

### Связь с массой

Масса как потенциальная энергия деформации:

```
E = mc² = σ_crit × A_critical
```

Откуда:

```
σ_crit = mc² / A_critical
```

### Критическая площадь

Для протона (m_p ≈ 1.67×10⁻²⁷ кг):

```
A_critical = m_p c² / σ_crit
```

Подстановка значений:
- m_p = 1.67×10⁻²⁷ кг
- c = 2.998×10⁸ м/с
- σ_crit = 10²⁴ Дж/нм² = 10³³ Дж/м²

```
A_critical = (1.67×10⁻²⁷) × (2.998×10⁸)² / 10³³
           = 1.50×10⁻¹⁰ / 10³³
           = 1.50×10⁻⁴³ м²
```

## Энергия прокола

### Определение

Энергия, необходимая для создания прокола в бране:

```
E_puncture = σ_crit × A_puncture
```

### Минимальная площадь прокола

Минимальная площадь прокола определяется планковской длиной:

```
A_puncture_min = l_Pl² = (1.616×10⁻³⁵)² м²
               = 2.61×10⁻⁷⁰ м²
```

### Минимальная энергия прокола

```
E_puncture_min = σ_crit × A_puncture_min
               = 10³³ × 2.61×10⁻⁷⁰
               = 2.61×10⁻³⁷ Дж
```

В единицах энергии:

```
E_puncture_min = 2.61×10⁻³⁷ / (1.602×10⁻¹⁹) эВ
               = 1.63×10⁻¹⁸ эВ
```

## Условия образования зеркальных зон (n < 0)

### Критическое условие

Показатель преломления становится отрицательным при:

```
σ(r, S) ≥ σ_crit
```

### Локальное натяжение

Локальное натяжение как функция расстояния:

```
σ(r, S) = σ₀ × exp[−k × S(r)]
```

где:
- σ₀ — фоновое натяжение
- k — warping параметр
- S(r) — масштабная координата как функция расстояния

### Условие критичности

```
σ₀ × exp[−k × S_crit] = σ_crit
```

Откуда:

```
S_crit = −(1/k) × ln(σ_crit / σ₀)
```

## Энергия извлечения из зон с n < 0

### Потенциальная энергия деформации

Энергия, запасённая в деформации браны:

```
E_deformation = ∫ (σ − σ_crit) dA
```

### Топологическая энергия инверсии

Энергия, связанная с топологической инверсией:

```
E_topological = (σ_crit × A_inversion) × f(S)
```

где f(S) учитывает фрактальную структуру.

### Общая энергия извлечения

```
E_extraction = E_deformation + E_topological
```

## Связь с другими параметрами

### Планковская энергия

Критическое напряжение связано с планковской энергией:

```
σ_crit ≈ E_Pl / l_Pl²
```

где:
- E_Pl = 1.956×10⁹ Дж (планковская энергия)
- l_Pl = 1.616×10⁻³⁵ м (планковская длина)

Проверка:

```
E_Pl / l_Pl² = 1.956×10⁹ / (1.616×10⁻³⁵)²
             = 1.956×10⁹ / 2.61×10⁻⁷⁰
             = 7.49×10⁷⁸ Дж/м²
```

Это значительно больше σ_crit, что указывает на необходимость учета warping эффекта.

### С учётом warping

С учётом экспоненциального подавления:

```
σ_crit_eff = σ_crit × exp(−2k|S|)
```

Для масштаба протона (|S| ≈ 11.2):

```
σ_crit_eff = 10³³ × exp(−2 × 0.1 × 11.2)
           = 10³³ × exp(−2.24)
           = 10³³ × 0.106
           = 1.06×10³² Дж/м²
```

| **E_puncture_min** | 2.61×10⁻³⁷ | Дж |
| **E_puncture_min** | 1.63×10⁻¹⁸ | эВ |

## Заключение

Расчёты показывают, что:
- Критическое напряжение браны имеет планковский масштаб
- Энергия прокола очень мала, но требует экстремальных условий
- Зеркальные зоны (n < 0) образуются при достижении критического натяжения
- Энергия извлечения связана с потенциальной энергией деформации
