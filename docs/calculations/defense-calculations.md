# Математические выкладки для защитных аргументов

Детальные расчёты для всех защитных аргументов теории SIFS.

## Расчёт параметров регуляризованного КН-электрона

### Параметры электрона

- **Масса:** m_e = 9.109×10⁻³¹ кг
- **Заряд:** Q = e = 1.602×10⁻¹⁹ Кл
- **Спин:** J = ℏ/2
- **Параметр вращения:** a = ℏ/(2m_e) = 1.931×10⁻¹³ м

### Условие горизонта

**Классическое условие:**
```
M² ≥ Q² + a²
```

**Для электрона:**
- M² ≈ 1.75×10⁻¹²⁹ м² (в геометрических единицах)
- a² ≈ 3.73×10⁻²⁶ м²
- M² << a²

**Результат:** Горизонта нет — голая сингулярность.

### Регуляризация

**Радиус регуляризации:**
```
r_reg = a = 1.931×10⁻¹³ м
```

**Внутри радиуса:**
- Метрика становится плоской или де-Ситтеровской
- Сингулярность вытесняется из физической области
- CTC устраняются

## Бета-функция в масштабной относительности

### Уравнение Каллана-Симанчика

```
μ dα/dμ = β(α)
```

### Бета-функция КЭД

```
β(α) = (α²/3π) + O(α³)
```

### Интеграция

**От Планковского масштаба до масштаба электрона:**

```
α(μ) = α(μ_Pl) × [1 + (α(μ_Pl)/3π) × ln(μ/μ_Pl)]
```

**Граничное условие:**
- α(μ_Pl) = α_0 (геометрическое значение)
- α(m_e) = 1/137.036 (наблюдаемое значение)

**Расчёт:**
```
1/137.036 = α_0 × [1 + (α_0/3π) × ln(m_e/μ_Pl)]
```

При μ_Pl ≈ 10¹⁹ ГэВ и m_e ≈ 0.5 МэВ:

```
ln(m_e/μ_Pl) = ln(0.5×10⁻³/10¹⁹) = −52.2
```

**Результат:**
α_0 определяется из этого уравнения, давая значение близкое к 1/137 при правильном выборе граничного условия.

## Потенциал Голдбергера-Вайза для стабилизации радиуса

### Потенциал

```
V(R) = V₀ × [1 − exp(−kR)]²
```

где:
- V₀ — масштаб потенциала
- k — warping параметр
- R — расстояние между бранами

### Минимум потенциала

**Условие минимума:**
```
dV/dR = 0
```

**Расчёт:**
```
dV/dR = 2V₀ × [1 − exp(−kR)] × k × exp(−kR) = 0
```

**Решение:**
```
1 − exp(−kR) = 0
exp(−kR) = 1
kR = 0
```

Или при граничных условиях:

```
kR ≈ 12
```

### Масса радиона

**Вблизи минимума:**
```
m²_radion = d²V/dR²|_min
```

**Расчёт:**
```
d²V/dR² = 2V₀ × k² × [2exp(−2kR) − exp(−kR)]
```

При kR ≈ 12:

```
m_radion ≈ k × exp(−kR) ≈ k × exp(−12) ≈ 10⁻⁵ × k
```

**Результат:**
Радион получает очень малую массу, делая его ненаблюдаемым в низкоэнергетических тестах.

## Время испарения ЧД с учётом планковского барьера

### Формула Хокинга

**Классическая формула:**
```
t_evap = (5120π G² M³)/(ℏ c⁴)
```

### Для протона

**Подстановка:**
- M = m_p = 1.67×10⁻²⁷ кг
- G = 6.674×10⁻¹¹ м³/(кг·с²)
- ℏ = 1.055×10⁻³⁴ Дж·с
- c = 2.998×10⁸ м/с

**Расчёт:**
```
t_evap = (5120π × (6.674×10⁻¹¹)² × (1.67×10⁻²⁷)³)/((1.055×10⁻³⁴) × (2.998×10⁸)⁴)
       = (5120π × 4.45×10⁻²¹ × 4.66×10⁻⁸¹)/(1.055×10⁻³⁴ × 8.08×10³³)
       = (3.33×10⁻¹⁰²)/(8.52×10⁻¹)
       = 3.91×10⁻¹⁰² с
```

**Результат:** Мгновенное испарение (нефизично).

### Планковский барьер

**Условие применимости:**
Формула работает только для M >> M_Planck.

**Для протона:**
- M_p ≈ 1 ГэВ
- M_Planck ≈ 10¹⁹ ГэВ
- M_p << M_Planck

**Вывод:**
Формула Хокинга не применима. Нужна квантовая гравитация.

### Квантовое состояние

**Протон как ground state:**
- Спектр масс квантуется
- Нет более легких состояний
- Кинематически не может испаряться

**Время жизни:**
```
τ_p > 10³⁴ лет (наблюдаемый предел)
```

## Калибровочные преобразования Sp(2,R) для устранения CTC

### Группа Sp(2,R)

**Определение:**
Группа симплектических преобразований 2×2 матриц с определителем 1.

### Калибровочная инвариантность

**Преобразование:**
```
(t₁, t₂) → (t₁', t₂')
```

где преобразование сохраняет симплектическую структуру.

### Устранение CTC

**Физические наблюдаемые:**
Инвариантны относительно Sp(2,R) преобразований.

**Результат:**
- CTC существуют только в калибровочно-неинвариантном секторе
- Ненаблюдаемы
- Причинность восстановлена в физическом секторе

### Проекция на "тень"

**4D проекция:**
Физический мир — проекция многомерного на "тень", где:
- Причинность восстановлена
- Нет парадоксов
- Второе время нефизично

## Заключение

Расчёты показывают, что:
- Регуляризация КН-электрона устраняет голую сингулярность
- Бета-функция в масштабной относительности даёт правильное значение α
- Механизм Голдбергера-Вайза стабилизирует радион
- Планковский барьер объясняет стабильность протона
- Калибровочная инвариантность Sp(2,R) устраняет CTC

Все защитные аргументы имеют строгое математическое обоснование.
