## 🚀 Быстрый старт

### Для физиков

Если вы хотите понять теоретические основы:

1. Начните с [Overview](theory/overview.md)
2. Изучите [Mathematics](theory/mathematics.md)
3. Ознакомьтесь с [расчётами](calculations/proton-mass.md)
4. Проверьте [наблюдательные данные](data/desi-2025.md)

### Для разработчиков

Если вы хотите внести вклад в код:

1. Клонируйте репозиторий
2. Изучите [структуру компонентов](visualizations/components.md)
3. Прочитайте [CONTRIBUTING.md](../CONTRIBUTING.md)
4. Создайте PR

### Для студентов

Рекомендуемый порядок изучения:

1. **Основы:** [Overview](theory/overview.md) → базовое понимание
2. **Геометрия:** [RS2 Geometry](theory/rs2-geometry.md) → как устроено 5D-пространство
3. **Физика частиц:** [Proton Mass](calculations/proton-mass.md) → почему протон стабилен
4. **Космология:** [DESI 2025](data/desi-2025.md) → современные данные
5. **Глубокая математика:** [Mathematics](theory/mathematics.md) → полный формализм
