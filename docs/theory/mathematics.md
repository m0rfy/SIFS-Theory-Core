# Mathematical Framework

Полный математический формализм теории SIFS.

## 📐 5D Metric

### Основная метрика

Warped 5D метрика Randall-Sundrum:

```
ds² = exp(−2k|S|) η_μν dx^μ dx^ν + dS²
```

где:
- `x^μ = (t, x, y, z)` — координаты на бране (4D)
- `S` — масштабная координата (5-я размерность)
- `k ≈ 0.1 M_Pl` — warping параметр
- `η_μν = diag(1, −1, −1, −1)` — метрика Минковского

### Компоненты метрического тензора

```
g_μν = exp(−2k|S|) η_μν
g_SS = 1
g_μS = 0
```

Определитель метрики:

```
√|g| = exp(−8k|S|)
```

## 🔄 Scale Transformations

### Инфинитезимальное преобразование

```
S → S + δS
x^μ → x^μ × exp(δS)
```

Поля преобразуются как:

```
φ(x, S) → exp(−Δ δS) φ(x × exp(δS), S + δS)
```

где `Δ` — масштабная размерность поля.

### Конечное преобразование

```
S → S + ΔS
λ → λ × exp(ΔS)
M → M × exp(−ΔS)
```

Длины масштабируются экспоненциально:

```
λ(S) = λ₀ exp(k|S|)
```

## ⚡ Action & Field Equations

### Действие в 5D

Полное действие включает bulk и braneworld компоненты:

```
S = S_bulk + S_brane

S_bulk = ∫ d⁴x dS √|g| (M₅³ R₅ + L_matter)

S_brane = ∫ d⁴x √|g₄| (−σ + L_brane)
```

где:
- `M₅` — 5D планковская масса
- `R₅` — скаляр Риччи в 5D
- `σ` — натяжение браны
- `L_matter` — материя в bulk
- `L_brane` — материя на бране

### Уравнения Эйнштейна

В 5D bulk:

```
R_AB − (1/2) g_AB R = (1/M₅³) T_AB
```

На бране (Israel junction conditions):

```
K_μν⁺ − K_μν⁻ = (1/M₅³) (T_μν − (1/3) T g_μν)
```

где `K_μν` — внешняя кривизна браны.

### Решение для warped метрики

Для AdS₅ bulk с кривизной `Λ₅ = −6k²`:

```
ds² = exp(−2k|S|) η_μν dx^μ dx^ν + dS²

k = √(−Λ₅/6) ≈ 0.1 M_Pl
```

Это решение уравнений Эйнштейна для:
- Отрицательная космологическая константа в bulk
- Положительное натяжение браны

## 🌊 Effective 4D Physics

### Редукция к 4D

Интегрируя по S, получаем эффективную 4D теорию:

```
S_eff = ∫ d⁴x √|g₄| M_Pl² R₄
```

где эффективная планковская масса:

```
M_Pl² = M₅³ ∫ dS exp(−4k|S|) = M₅³/(2k)
```

### Эффективная гравитационная константа

На масштабе `|S|`:

```
G_eff(S) = G₅ exp(−2k|S|)
```

где `G₅ = 1/M₅³` — 5D гравитационная константа.

Для больших `|S|` (макроскопические масштабы):

```
G_eff → 6.674 × 10⁻¹¹ m³/(kg·s²)
```

## 🔢 Kaluza-Klein Decomposition

### KK-разложение полей

Поле φ в 5D разлагается:

```
φ(x^μ, S) = Σₙ φₙ(x^μ) ψₙ(S)
```

где `ψₙ(S)` удовлетворяют:

```
[−∂²_S + V(S)] ψₙ = mₙ² ψₙ
```

### KK-массы

Для warped метрики:

```
mₙ² ≈ (πn/|S_max|)² exp(2k|S|)
```

Zero-mode (n=0):

```
ψ₀(S) ∝ exp(2k|S|)
```

локализован на бране с малым `|S|`.

## 🌀 Fractal Structure

### Лог-периодичность

Корреляционная функция:

```
C(r, r') = C₀ cos(ω ln(r/r'))
```

где `ω ≈ 2π` — фрактальная частота.

### Фрактальная размерность

На масштабе `r`:

```
D_f(r) = 3 + δ(r)

δ(r) = δ₀ exp(−r/ξ)
```

где `ξ` — корреляционная длина.

Для больших масштабов: `D_f → 3` (восстановление стандартной геометрии).

## 🌌 Dark Energy

### Эффективная космологическая константа

```
Λ_eff = (S_horizon/A_horizon) exp(−2k|S_global|)
```

где:
- `S_horizon` — энтропия горизонта
- `A_horizon` — площадь горизонта
- `|S_global|` — глобальная масштабная координата

### Эволюция

```
S_global(z) = S₀ + δS × z/(1+z)

Λ_eff(z) ∝ exp(−2k S_global(z))
```

Equation of state:

```
w(z) = −1 + (2k δS)/(3(1+z)²)
```

## 🕳️ Black Hole Microphysics

### Kerr-Newman метрика

Для вращающейся заряженной ЧД:

```
ds² = −Δ/ρ² (dt − a sin²θ dφ)² 
      + sin²θ/ρ² ((r² + a²)dφ − a dt)²
      + ρ²(dr²/Δ + dθ²)
```

где:
```
Δ = r² − 2Mr + a² + Q²
ρ² = r² + a² cos²θ
```

### Эффективный горизонт

На масштабе `|S|`:

```
r_eff = r_s exp(k|S|/2)
```

Для протона (`|S| ≈ 11.2`):

```
r_eff ≈ (2.48 × 10⁻⁵⁴ m) × exp(5.6) ≈ 230 fm
```

(в bulk, ненаблюдаемо в 4D)
