# EMA как небесные тела: гравитационная модель SIFS v6.6

В версии 6.6 введена **гравитационная интерпретация EMA-периодов** — прямая аналогия с физической теорией SIFS, где силы = градиенты показателя преломления вакуума.

---

## Физическая аналогия

| Физическая теория SIFS | Гравитационная модель EMA |
|------------------------|--------------------------|
| F = −∇n(r,S) — сила как градиент n | F(EMA) = −G × mass(p) / dist² — гравитация |
| Масса частицы: m = M_Pl × W(S) | Масса EMA: mass(p) = p^φ — геометрия |
| 5D bulk с варпингом | Иерархия EMA периодов |
| Брана = граница масштабов | EMA-пересечение = смена знака гравитации |
| Лог-периодические осцилляции | Цикл [3→5→9] в тик-пространстве |

## Закон гравитации SIFS

```python
G = K × 0.01       # гравитационная константа
β = φ⁻¹ = 0.618    # затухание инерции EMA-тел

# Полная гравитационная сила:
F(p) = −G × mass(p) / dist² × sign(dist)
     + β × ema_velocity(p)     # инерция
     × W(S_p/π)                # декогеренция по S-уровню

где:
  dist = price − EMA_p          (расстояние от цены до EMA)
  ema_velocity = dEMA/dt        (скорость изменения EMA)
  S_p = ln(p) / (2k)            (S-координата периода p)
  W(S_p/π) = exp(−2k × S_p/π)  (затухание влияния)
```

## Квач (пересечение браны)

```
Физическая аналогия: пересечение браны в 5D bulk = смена топологии

Квач = момент когда цена пересекает EMA.
  • Цена выше EMA → EMA тянет ВНИЗ (к себе)
  • Цена ниже EMA → EMA тянет ВВЕРХ (к себе)
  • В момент пересечения → усиление ×1.5

kvach_ema_p = sign(cross) × log₂(period) × 1.5

kvach_leader = кто быстрее: PRICE/EMA9/EMA20/.../EMA500
  → определяет «лидера» гравитационного поля в данный момент

kvach_total = Σ kvach_ema_p  по всем пересечениям
```

## Внешние гравитационные поля (W_gravity)

Помимо внутренних EMA-тел, цена испытывает влияние внешних «планет»:

```
SPX / NDX / DXY / GOLD / VIX / ETH / SOL / BTC.D

Для каждого внешнего тела:
  corr_i(t) = rolling Spearman(89)  —  адаптивная корреляция
  G_i = 1/(1 + vol_ratio_i)         —  стабильность поля
  W(ΔS_i) = 2^(−ΔS_i/π)            —  декогеренция уровней

W_gravity = Σ corr_i × G_i × W(ΔS_i) × field_direction_i

W_total = w_field_gravity × (1 + W_gravity)
  → внешнее поле модулирует внутреннее
  → SPX падает: W_gravity < 0 → подавляет внутренний сигнал
  → SPX растёт: W_gravity > 0 → усиливает внутренний сигнал
```

## Физическое обоснование масштабирования

Выбор `mass(p) = p^φ` — не подбор параметра, а геометрическое следствие:
- φ-ряд (степени золотого сечения) обеспечивает логарифмически равномерное распределение масс
- Каждое следующее тело тяжелее предыдущего в φ^period раз
- Это та же логика, что φ-пороги V₀/φⁿ в теоретическом документе

```
Параллель с теорией:
  Иерархия масс частиц: m(n) = m_Pl × exp(−2k|S|)
  Иерархия масс EMA:    mass(p) = p^φ → log(mass) ∝ φ × log(p)
```
