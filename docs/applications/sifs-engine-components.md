# Архитектура Score: компоненты W1–W9

Score — итоговая мера структурного состояния системы на данном таймфрейме.
Вычисляется как взвешенная сумма 9 компонентов.

---

## Итоговая формула Score

```
Score_raw = Σᵢ Wᵢ × Cᵢ     i = 1..9

Score = Score_raw / Нормировочный_фактор

Порог значимости: |Score| > GM = (π+φ)/2 ≈ 2.38
```

---

## Компоненты W1–W9

### W1 — Объём × φ⁻¹ (информационный поток)

```
Физическая аналогия: скорость передачи информации = c² в метрике ds²
Формула: W1 = (V_curr/V_median − 1) × φ⁻¹ × sign(direction)
  V_median = медиана объёма за 89 свечей

Интерпретация:
  W1 >> 0 → объём выше нормы в направлении движения → «истинный» тик
  W1 << 0 → объём аномально низкий → «эхо» или шум

Вес в Score: наибольший (часто ±3.0 — ±5.0 при аномалиях)
```

### W2 — Ускорение × Ψ₂ (инерция движения)

```
Физическая аналогия: изменение кривизны геодезической
Формула: W2 = (momentum_curr − momentum_prev) × Ψ₂
  где Ψ₂ = 9/(2π) ≈ 1.432

Интерпретация:
  W2 > 0 → импульс нарастает → движение ускоряется
  W2 < 0 → импульс ослабевает → разворот или консолидация

Особо значим: выброс W2 (|W2| > 3) на одном ТФ при W2≈0 на соседних
             → локальная аномалия, не структурный сигнал
```

### W3 — OBV (сумма объёмов с учётом направления)

```
Физическая аналогия: накопленный заряд на бране
Формула: W3 = sign(OBV_delta) × min(|OBV_delta/OBV_std|, 1)
  Нормировано в [-1, +1]

Интерпретация:
  W3 > 0 → покупки преобладают по объёму
  W3 < 0 → продажи преобладают по объёму
  |W3| ≈ 0 → нейтральное давление
```

### W4 — VWAP (отклонение цены от справедливой)

```
Физическая аналогия: отклонение от геодезической (центра масс)
Формула: W4 = (price − VWAP) / (ATR × k_norm)

Интерпретация:
  W4 > 0 → цена выше справедливой стоимости → тяготение вниз
  W4 < 0 → цена ниже справедливой стоимости → тяготение вверх
  |W4| ≈ 0 → равновесие (точка Лагранжа между VWAP и ценой)
```

### W5 — FER (Fractal Expansion Ratio, расширение структуры)

```
Физическая аналогия: тёмная энергия — скорость расширения
Формула: FER = σ_price(T) / σ_price(T_base)
  FER > 1 → расширение структуры (expansion)
  FER < 0.3 → сжатие структуры (contraction)

Значение в Score:
  Если FER < 0.3 (Contraction) → система накапливает давление
  Если FER > 1.5 (Expansion) → система в высвобождении
```

### W6 — Fractal Dimension D (сложность структуры)

```
Физическая аналогия: фрактальная размерность пространства-времени
Формула: D = log(N) / log(1/r)  (алгоритм box-counting или Hurst)
  D ∈ [1.0, 2.0]
  D = 1.0 → чистый тренд (линия)
  D = 1.5 → случайное блуждание
  D = 2.0 → полный хаос (площадь)

Значение в Score:
  D ∈ [1.1, 1.3] → структурированный тренд → сигнал надёжен
  D > 1.5 → шумовой режим → сигналу не доверять
```

### W7 — Шум (волатильность самого Score)

```
Формула: W7 = Std(Score, 21) / Score_max_historical
  (начиная с v6.2: реальная волатильность, не константа)

Интерпретация:
  W7 < 0.1 → Score стабилен → сигнал чистый
  W7 > 0.3 → Score нестабилен → высокий шум, снизить доверие
```

### W8 — MTF варпинг (согласие между таймфреймами)

```
Физическая аналогия: запутанность между бранами (Entanglement)
Формула: W8 = Σ W(a→b) × Score(b) × corr(Score_a, Score_b, lag)
  Суммирование по всем парам (ТФ_a, ТФ_b)
  W(a→b) = exp(−2k × |S(ТФ_a) − S(ТФ_b)|)

Интерпретация:
  W8 > 0 → все ТФ согласованы → сигнал подтверждён на структурном уровне
  W8 < 0 → ТФ конфликтуют → «сэндвич», рынок не определился
  Рассинхронизация ≡ предвестник крупного движения
```

### W9 — Лог-периодическая фаза φ_S (положение в цикле, с v6.2)

```
Физическая аналогия: лог-периодические осцилляции δS ≈ 2π (предсказание теории)
Формула: φ_S = fractional_part(cum_ticks / T₃)
  T₃ = 9 = floor(π²) — полный цикл
  φ_S ∈ [0, 1)

Интерпретация:
  φ_S ∈ [0, 1/3)  → Истина (3 тика, первое проявление)
  φ_S ∈ [1/3, 5/9) → Эхо (5 тиков, подтверждение)
  φ_S ∈ [5/9, 1)  → Коллапс (приближение к 9)

W9 = cos(2π × φ_S)  — непрерывная мера положения в цикле
```

## Гравитационная модель EMA (v6.6)

Начиная с версии 6.6, EMA-периоды моделируются как **небесные тела**:

```python
mass(p) = p^φ                     # масса тела — геометрия, не подбор

EMA9:    9^φ  ≈     35  # Луна
EMA20:   20^φ ≈    127  # Планета
EMA50:   50^φ ≈    561  # Звезда
EMA89:   89^φ ≈  1 426  # Гигантская звезда
EMA200: 200^φ ≈  5 286  # Галактическое ядро
EMA500: 500^φ ≈ 23 282  # Сверхмассивная чёрная дыра

Закон притяжения:
F(p) = −G × mass(p) / dist² × sign(dist) + β × ema_velocity
  dist = price − EMA_p  (расстояние от цены до тела)
  EMA500 притягивает в 665× сильнее EMA9 на том же расстоянии
  β = φ⁻¹ = 0.618  (затухание инерции)

Квач: пересечение EMA = смена знака гравитации × log₂(period) × 1.5
```
