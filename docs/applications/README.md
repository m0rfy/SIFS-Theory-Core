# SIFS: Применения математического ядра

Этот раздел документирует применение математического ядра **Scale-Invariant Fractal System (SIFS)** к реальным задачам — от анализа финансовых рынков до универсального детектора структурных переходов в любых иерархических системах.

> **Важно:** Документы здесь описывают **аналитический инструмент** (SIFS System), использующий ту же математическую форму, что и физическая теория [SIFS Theory (Spacetime)](../related-projects.md). Это осознанная аналогия, не строгий физический вывод.

## Связь с физической теорией

| Физическая теория SIFS (Spacetime) | Аналитический инструмент SIFS (System) |
|------------------------------------|----------------------------------------|
| ds² = exp(−2k\|S\|)·η_μν dx^μdx^ν + dS² | W(n) = exp(−2k\|n\|) — компрессия информации |
| S — масштабная координата (5-е изм.) | S — таймфрейм: S(T) = ln(T_min/T_base)/(2k) |
| exp(−2k\|S\|) подавляет массу частицы | W(n) подавляет видимость тиков соседних ТФ |
| Лог-периодические осцилляции δS≈2π | Цикл тиков [3→5→9], период ≈ 2π |
| F = −∇n(r,S) — все силы как градиент n | Score = −∇(информационного потенциала) по S |
| Протон = effective horizon (Kerr-Newman) | Горизонт коллапса = Φ_dark, магнит для цены |
| Голографическая запутанность ER=EPR | Entanglement(ТФ_a, ТФ_b) — синхронизация |
| Энергосохранение: 3D→4D→5D | 80% S+1 / 15% S−1 / 5% Φ_dark |
