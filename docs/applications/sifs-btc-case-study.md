# Кейс: Bitcoin 2009–2026 через призму SIFS

Применение SIFS System к полной истории Bitcoin (2009–2026).
Анализ месячных закрытий с идентификацией ключевых структурных переходов (коллапсов).

> *Термин «коллапс» в SIFS не означает «крах рынка» — это структурный переход (смена нулевой точки S=0). Движение вверх тоже является коллапсом.*

---

## Базовые параметры

```
Уровень наблюдения: S=2 (месячный ТФ)
1 тик = медианный логарифмический шаг: ln(P_t / P_{t-1}) ≈ 0.12 ln-единицы
Коллапс: накопленные тики от S=0 ≥ 9 (= floor(π²))
```
