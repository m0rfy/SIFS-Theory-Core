# SIFS System: от теории к инструменту

**SIFS System** — универсальный детектор структурных переходов в иерархических системах.
Основан на той же математической форме, что и физическая теория [SIFS Theory (Spacetime)](../related-projects.md).

---

## Центральная идея

> **SIFS — не детектор тренда. SIFS — детектор структурного разрыва.**

Система выявляет **логические разрывы** между тремя слоями реальности:
- **Block I** → Нарративный шум / Глобальный контекст
- **Block II** → Институциональные потоки
- **Block III** → Фактическое развёртывание капитала

**Истина всегда локализована в Block III.**

## Наблюдатель как часть системы

Ключевая идея из физической теории: **выбор масштаба наблюдения — это выбор координаты S**.

```
Физика:           наблюдатель в 5D-bulk выбирает брану → видит разную массу
Рынок:            трейдер выбирает таймфрейм → видит разную структуру
Общий принцип:    данные не изменились — изменилась точка наблюдения
```

Это означает, что **«противоречие» между 1m и 1d** — не ошибка, а информация о разных уровнях S.

## Фундаментальные константы (SIFS System)

```
π  = 3.14159265...   базовая мера минимального движения
φ  = 1.61803398...   золотое сечение (оптимальная иерархическая упаковка)

Ψ₁ = 9/π   ≈ 2.8648  скейлинг между уровнями S
Ψ₂ = 9/π/2 ≈ 1.4324  скейлинг внутри уровня S
GM = (π+φ)/2 ≈ 2.3799 золотая середина (порог значимости Score)

k    = 1/(π×Ψ₁) ≈ 0.1111   параметр затухания (аналог RS-warping)
W(n) = exp(−2k|n|)           варпинг-функция

Значения W(n):
  W(0) = 1.000   W(1) = 0.800   W(2) = 0.641
  W(3) = 0.513   W(4) = 0.411   W(5) = 0.329
  W(6) = 0.263   W(7) = 0.211   W(8) = 0.169
```

**Принцип иррациональности:** π и φ трансцендентны. Их произведения никогда не совпадают точно с целыми числами → сетка уровней никогда не «захлопывается» в периодичность → система остаётся фрактальной на любой глубине.

## Отображение таймфреймов на координату S

Строгая формула (введена в v6.2):

```
S(T) = ln(T_минут / T_min_минут) / (2k)

Для набора {1m, 3m, 5m, 15m, 30m, 1h, 2h, 4h, 12h, 1d}:
  S(1m)   = 0.00     S(3m)   = 4.95
  S(5m)   = 7.24     S(15m)  = 12.19
  S(30m)  = 14.49    S(1h)   = 16.48 (вариант: 18.59 по другой базе)
  S(4h)   = 24.28    S(1d)   = 32.40

Варпинг между ТФ_a и ТФ_b:
  n(a,b) = |S(ТФ_a) − S(ТФ_b)|
  W(a→b) = exp(−2k × n(a,b))

Пример: W(1m→5m) = exp(−2k × 7.24) = 0.200
  (NOT W(1) = 0.800 — это W(4) в дискретной шкале)
```

## Цикл [3→5→9] и коллапс

Цикл тиков в SIFS соответствует лог-периодическим осцилляциям физической теории (δS ≈ 2π):

```
3 тика → «Истина» (первое направление проявлено)
5 тиков → «Эхо»   (подтверждение или ложный сигнал)
9 тиков → «Коллапс» (структурный переход, S=0 пересчитывается)

9 = floor(π²) ≈ 9.87
3 ≈ π, 5 ≈ π+φ, 9 ≈ π²

δS_цикл = 2π ≈ 6.283 — период осцилляций (как в CMB предсказании теории)
```

Коллапс не означает «конец» — это **смена нулевой точки S=0**. Энергия сохраняется:
- 80% → уровень S+1 (масштаб выше)
- 15% → уровень S−1 (масштаб ниже)
- 5% → Φ_dark (память у горизонта)

## Ограничения и честные предупреждения

- Отображение TF→S (дискретное vs непрерывное) — активная область исследования, выбор влияет на силу W
- Score не является прогнозом цены — это мера структурного состояния системы
- Порог значимости GM = (π+φ)/2 ≈ 2.38 — эмпирически обоснован, не строго выведен
- Вероятности P(%) в анализе — из логистической калибровки, требуют регулярного обновления
