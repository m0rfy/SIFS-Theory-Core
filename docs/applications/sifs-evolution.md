# Эволюция SIFS System: v5 → v6.6

Хронология ключевых изменений аналитического инструмента SIFS.
Каждая версия отвечала на конкретную проблему или добавляла новую физическую аналогию из теории.

---

## SIFS v5.0 (базовая, ~2025)

**Принцип:** Цикл тиков [3→5→9], варпинг W(n) = exp(−2k|n|), фиксированное назначение S.

Компоненты Score: W1 (объём), W2 (ускорение), W3 (OBV), W4 (VWAP), W5 (FER), W6 (Fractal Dimension), W7 (шум).

**Ключевая находка:** Все ТФ могут быть в «коллапсе» одновременно — не потому что данные одинаковые, а потому что каждый ТФ имеет свой порог на своём уровне S.

---

## SIFS v6.0 (2025)

**Добавлено:**
- Формализация определения тика (3 обязательных условия)
- Вес тика по объёму: 0.5 / 1.0 / 1.5
- Пороги [3→5→9] как дискретные метки на непрерывной шкале
- Механизм сброса S=0 при превышении порога

**Проблема:** S назначается произвольно (n=0,1,2...) → разница в варпинге между «соседними» ТФ в 4× отличается от документа.

---

## SIFS v6.1 (2025)

**Добавлено:**
- **Компрессия тиков при наблюдении между уровнями** через W(n)
- Пороги 3/5/9 применяются только к тикам своего уровня S
- При взгляде на соседний уровень тики сжимаются/растягиваются через варпинг

**Ключевое исправление:** Объяснено, почему на BTC все ТФ показывали «коллапс» одновременно — каждый был в коллапсе на своём уровне.

---

## SIFS v6.2 (2026)

**Добавлено:**
- **Формальное отображение TF→S:** `S(T) = ln(T_min / T_base) / (2k)` — математически точное
- **W9: Лог-периодическая фаза φ_S** — непрерывная фаза цикла вместо дискретного счётчика
- **W_time:** длительность тика относительно исторической нормы влияет на Score
- **W7 исправлен:** реальная волатильность Score вместо константного шума
- **Алгоритм выбора S=0** формализован (4 критерия с весами)
- **Инвалидация прогноза** автоматически (3 триггера)
- **Калиброванные вероятности** через логистическую функцию

```
S(1m)=0.00   S(5m)=7.24   S(15m)=11.94
S(1h)=18.59  S(4h)=24.28  S(1d)=32.40
```

## SIFS v6.3 (2026)

**Добавлено:**
- **W_field:** EMA создают поля притяжения, цена между двумя EMA = точка Лагранжа
- **W_spin:** внутреннее состояние свечи (положение тела в диапазоне high-low)
- **Сохранение энергии коллапса:** 80% → S+1, 15% → S−1, 5% → Φ_dark
- **Запутанность таймфреймов (Entanglement):** корреляция Score между ТФ с лагом
- **Волновая функция тика Ψ:** вектор вероятностей 4 исходов
- **Геодезическая цены:** интеграл Score → конкретная ценовая цель

Таблица соответствий:

| Физическая теория | Инструмент v6.3 |
|---|---|
| Протон = effective horizon (Kerr) | Горизонт коллапса = Φ_dark |
| Голографическая запутанность ER=EPR | Entanglement(ТФ_a, ТФ_b) |
| Волновая функция → коллапс | Ψ(tick) → 4 исхода |
| Геодезическая в 5D | Интеграл Score → Price(t+1) |

## SIFS v6.4 (2026)

**Добавлено:**
- **τ_S — собственное время уровня S** (время не универсально — разное на каждом ТФ)
- **Volume-Time (VT-ось):** каждая свеча имеет «объёмный вес» вместо равного шага
- **Асимметрия варпинга:** W_up ≠ W_down (рынки падают быстрее чем растут)
- **ω_S — собственная частота уровня S** (естественный ритм полного цикла [3→9])
- **W_corridor:** BB-velocity + BB-acceleration + межмасштабное синхронное сжатие
- **Детектор режима рынка:** Тренд / Флет / Разворот (устраняет инверсию Spearman)
- **Φ_dark в тик-времени:** память горизонтов затухает по числу тиков, не по UTC

```python
α_time = exp(2k) ≈ 1.250  # асимметрия времени
W_up(n)   = exp(−2k|n|)              # прямое движение
W_down(n) = exp(−2k|n| × α_time)    # обратное движение быстрее
```

## SIFS v6.6 «Гравитационная» (2026)

**Принципиальное изменение:** EMA-периоды = небесные тела с гравитацией.

```
EMA9=Луна, EMA20=Планета, EMA50=Звезда,
EMA89=Гигантская звезда, EMA200=Галактика, EMA500=ЧД
```

**Добавлено:**
- `mass(p) = p^φ` — масса тела геометрически выведена, не подбирается
- `F(p) = −G × mass(p) / dist² + β × ema_velocity` — закон притяжения
- **W_gravity:** внешние гравитационные поля (SPX, DXY, GOLD, VIX и др.)
- **Квач иерархический:** `kvach_leader` — кто быстрее из всех EMA
- **Адаптивные корреляции:** rolling Spearman(89) для каждого внешнего тела
- **Стресс-асимметрия:** ρ→ρ_stress при высоком vol_ratio

Подробнее: [sifs-gravity-model.md](sifs-gravity-model.md)
