# Чеклист адаптивности: Проверка всех страниц на разных размерах экранов

**Задача**: T066 [US5]  
**Цель**: Убедиться, что все страницы работают корректно на всех размерах экранов (320px-1920px+)  
**Критерий успеха**: 100% страниц работают корректно на всех размерах экранов

## Размеры экранов для проверки

- 320px (мобильный, очень маленький)
- 640px (мобильный, средний)
- 768px (планшет, портрет)
- 1024px (планшет, альбом / ноутбук)
- 1280px (десктоп, средний)
- 1920px (десктоп, большой)

## Критерии проверки на каждом размере

Для каждой страницы проверить:

1. **Навигация работает**
   - [ ] Orbital Dock виден и доступен
   - [ ] Меню открывается/закрывается корректно
   - [ ] Все ссылки кликабельны
   - [ ] Нет перекрытия элементов навигации

2. **Текст читаем**
   - [ ] Размер шрифта достаточен (минимум 14px для основного текста)
   - [ ] Контрастность достаточна
   - [ ] Текст не обрезается
   - [ ] Переносы строк корректны
   - [ ] Максимальная ширина текста соблюдена (800px для основного текста)

3. **Графики масштабируются**
   - [ ] Графики видны полностью
   - [ ] Графики не выходят за границы контейнера
   - [ ] Подписи осей читаемы
   - [ ] Интерактивные элементы доступны

4. **Таблицы прокручиваются**
   - [ ] Таблицы не ломают layout
   - [ ] Горизонтальная прокрутка работает (если нужно)
   - [ ] Заголовки таблиц видны
   - [ ] Ячейки читаемы

5. **Функциональность сохранена**
   - [ ] Все кнопки кликабельны (минимум 44x44px на мобильных)
   - [ ] Формы работают корректно
   - [ ] Интерактивные элементы доступны
   - [ ] Анимации не мешают взаимодействию

## Инструкции по тестированию

### Использование Browser DevTools

1. Открыть приложение в браузере (Chrome, Firefox, Safari, Edge)
2. Открыть DevTools (F12 или Cmd+Option+I)
3. Включить Responsive Design Mode (Ctrl+Shift+M или Cmd+Shift+M)
4. Выбрать размер экрана из списка или ввести вручную
5. Проверить все критерии для данного размера
6. Записать результаты в чеклист

### Альтернативные методы

- Использовать онлайн-инструменты для тестирования (BrowserStack, Responsively)
- Использовать реальные устройства (если доступны)
- Использовать эмуляторы устройств

## Отчёт о проблемах

### Формат записи проблемы

```
Страница: [название страницы]
Размер экрана: [размер в px]
Проблема: [описание проблемы]
Критичность: [высокая/средняя/низкая]
Шаги для воспроизведения:
1. [шаг 1]
2. [шаг 2]
3. [шаг 3]
Ожидаемое поведение: [что должно быть]
Фактическое поведение: [что происходит]
```

### Примеры проблем

```
Страница: HomePage
Размер экрана: 320px
Проблема: Orbital Dock перекрывает заголовок страницы
Критичность: высокая
Шаги для воспроизведения:
1. Открыть HomePage на экране 320px
2. Прокрутить страницу вверх
Ожидаемое поведение: Orbital Dock не должен перекрывать контент
Фактическое поведение: Orbital Dock перекрывает заголовок "SIFS Theory"
```
